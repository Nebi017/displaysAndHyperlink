The display Property

display:inline

display: inline in CSS makes an element display inline with the surrounding content, without starting a new line or affecting the layout height



display:inline-block

inline-block in CSS allows an element to be styled as an inline element while retaining the ability to set width and height properties.



display:block

display: block; in CSS makes an element block level element , causing it to take up the full width available and start on a new line.
